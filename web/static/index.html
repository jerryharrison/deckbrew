From a riptide laboratory deep below the underground sea, we're toiling away on creating the greatest deck building website ever to see the light of day. While we work, check out our awesome Magic: The Gathering API.

Changelog

June 21st, 2015

Dragons of Tarkir and Fate Reforged have both been added. Images are now powered by Gatherer, as Wizards shut down mtgimage.com.

July 8th, 2014

Magic 2015 has been added to the website. Multiverse IDs can now also be specified using the 'm' URL parameters when filtering cards.
